# 歡迎來到 睡死後之兵 Pawn After Slumber!

睡死後之兵是一款基於棋類殘局解謎的無雙棋類遊戲，
在遊戲中你扮演著一個守護符，
在人們睡著後清除他們腦內的煩惱。
遏制惡性思考無限制的成長，
用爽快的連殺爆破來對抗它們吧！

# 遊戲特色介紹：

# 連殺

當你殺死一個目標時，可以立刻再次行動一次，直到連殺中斷！

# 複製

當你擊殺敵方目標時，你將複製他的能力，並繼承他的攻擊範圍！

# 進化的敵人

每過一回合，敵人都會再次強化，並且不斷湧現而出！
