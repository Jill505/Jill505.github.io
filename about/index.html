# 遊戲創作

主要做 2D 的，3D 的因為萬象鎖問題一直不敢碰，四元數也因為我是文組所以一直對他提不起興致
我十分享受從開始從流程圖、繪製美術資源一直到寫程式的每一個環節
但故事創作是我在遊戲創作的過程中最享受的一部份
透過互動的方式講述一段故事，透過層層嵌合的方式來講述一個理念是我一直所追尋的藝術

# 程式

數學各種花式被當，但不知道為什麼打 code 的時候都寫的出來 (我是寫程序的不是數學家.jpg)

主 C#(Unity)，偶爾寫點 JS 和 Python
之前為了考 APCS 和 UE5 學過 C++，不過荒廢很久了 冏

# 繪畫

平常外出時畫的比較多，假日和放學一整天坐到電腦前不是在 Visual Studio 就是在召喚峽谷
電繪是挺荒廢，很偶爾才會畫一下
一些搞笑的小廢圖、四格、素描挺高產
速寫則是每天都至少畫個五六張維持手感
幾個月前剛入坑 stable-diffusion，還在調適 AI 輔助繪畫的介入程度，但總感覺還是沒有基本功練扎實來得有用
