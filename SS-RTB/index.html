# 返祝 - Return The Blessing

在這個世界中存在著一個秘密：
透過某種神祕的儀式，人們可以把記憶消除掉、獻給「神明」，換來相應的祝福

有天，有個無名少女在棺材中醒來，腦袋一片空白，手裡卻握著強大的各種祝福
失去了所有的記憶的她感到十分的恐慌
唯一想要做的事，只有把自己的記憶奪回來！

於是她踏上了征討神明的旅途，擊敗神明、返還祝福

在一次又一次的失敗、死亡、重複失憶與找回碎片記憶的過程中，她也在逐漸面對一個問題：
「如果每一次拿回記憶全貌的我都選擇拋棄這段記憶，那麼這段看似只會給我帶來痛苦的記憶究竟有沒有追尋的必要？」

# 特色系統介紹

返祝是一款特殊的 Rogue like 遊戲
在一般的遊戲中，玩家的能力都是逐漸增長的
但在返祝中，玩家隨著雅菲不斷的拋棄能力、拾回記憶，能力反而會越來越弱

玩家必須仔細研究自己的打風與規劃 boss 的攻略順序
才能達到最完全的「返祝」
