# 遊戲創作的契機

當時我還是幼兒園，在第一次在網路上玩到「賽爾號」時
我首先的反應是驚艷：居然能夠將戰鬥的過程進行數值化，真的是太天才了！
接著第二個反應便是：他能做，我為什麼不能做呢？
抱著這樣的想法，第二天來到了幼兒園，我從廢紙堆中創了我自己的「賽爾號」並邀請同學們一起遊玩，雖然那個遊戲有的只是幾張廢紙和幼兒園水平的美術圖
但就在那一刻我第一次知道了：「原來看到玩家的笑容，比玩遊戲還要快樂」
也在那時我下定了決心：「我要做遊戲！」

# 相信基礎，不信一步登天

在升上國中後，我意外得到了一台被回收的電腦，這讓我萌生了製作電子遊戲的想法，但我深知，做遊戲有一定的技術門檻，以當時的實力不可能一蹴而就，一步一步老老實實的把基礎練習好才是夢想大廈最堅實的地基
為此，我將國中三年分成了三個階段，第一年企劃，第二年美術，第三年程式，聽起來有些自吹自擂，但這三個分類是我在國一的時候自己盯著螢幕想出來的，當幾年後知道開發職務分工也是如此時真的十分驚訝

而在國中三年間，我得到了許多來自家人、朋友的幫助，並在前輩們的指點下紮實的提升自己的水平
多虧了作為遊戲企劃的大舅、配音員的小舅、插畫家的舅媽、工程師的姨丈，以及一群對遊戲製作有熱誠的網友。
在這三年，我的美術水平成長如圖：![](insert picture)
程式也從鍵盤都不會打進步到學會基本的物件導向
企劃方面我則透過對每一款我玩到的遊戲去思考「為什麼這款遊戲會好玩？」以及撰寫文章來練習

# 豐富的遊戲開發經驗

在高中的這三年間，我共製作了六款有完整 Game Loop 的遊戲
他們分別是賭夜、六家廢土、返祝、佑可別睡了、閃子彈、業新聞 //(未來附上載點)
在製作的過程中，我大量積累了不同種類遊戲從研發期到開發期 的經驗，對於製程的流程與開發節奏的掌握有一定的經驗與心得。

# 程式面

我學會了將一個複雜的需求分解成許多的小步驟去逐步破解、在大型專案與多人協作中培養了程式的大局觀與分工經驗積累、我甚至為了能在學校時用紙筆也能夠撰寫所需的功能，把 Unity 中常用的方法集都背了下來。
而我在對於程式碼有不理解時也勇於在社群發問、寫信請教插件原作者，真的不懂時也會去請教學校老師，而在 ChatGPT 出現後，我也經常使用它來解惑程式相關的問題。

# 美術面

秉持著「走到哪畫到哪」的精神，我每天都進行著持續的速寫練習，每隔兩三天也一定會練習幾張素描，在家中我也練習了許多電繪與臨摹。
在練習的同時，我也持續的在吸收繪畫的相關知識
因為我並非是美術專科的學生，所以我在知識的吸收上需要更加的努力，無論是觀看網路上的教學影片或是閱讀書籍，我都付出比他人更多的心力與時間。
無論是 Krenz 與 Proko 的教學影片、羽山純一「人體動態速寫」，還是 George B Bridgman「Complete Guide To Drawing From Life」、Stepphen Rogers Peck「藝用人體解剖」、Lise Herzog「繪畫的基本」，我都拿出了十二分的努力去理解與學習，並同時搭配練習來使學到的理論能夠落地。

# 企劃面

除了在玩遊戲時會分析遊戲的製作邏輯並持續追蹤「巴哈姆特」的報導以得到最新的業界消息外，我還積累了許多專案與團隊管理相關的經驗，在這三年內，我帶領過從三人到至最多八人一起開發過，雖然不是每次的開發都一帆風順，但我總是以積極的態度去面對團隊中的每個矛盾與衝突，同時為每位團隊成員分配難度適中的工作內容、期限，而在每個禮拜的固定會議時間中也鼓勵成員們發表意見。
而受益於同時進行著程式與美術方面的學習，在與不同領域的同學們溝通時，我可以作為不同專業之間的溝通橋樑，並能在規劃時就將程式與美術的可行性、時間成本、能力範疇一併考慮，並在分配工作時能給出明確的規格。

# 真心熱愛而享受著開發的過程

在開發的期間雖然有許多的困難與挫折，但我沒有放棄過創作，曾經有無數個夜晚在昏暗的房間內研讀著原文的插件 Document 、練習枯燥的排線、計算各種遊戲內的參數。
起初我覺得這些工作非常的累人且枯燥，但在創作的過程中，我發現了這些工作的有趣之處，繪畫產生的激情、程式撰寫的「靈光一閃」時刻、計算參數的醍醐味，原本我只是為了他人的笑容而努力，但當我回過神來才發現，我對遊戲開發的愛已經滲透了製程的每一個角落，我是真心喜歡著遊戲開發。

# 積極參加各種活動與展覽

在埋頭製作的同時，我也積極的參加著各種活動、講座與展覽，從參展第三、四屆的 TSGDF 到參與巴哈姆特自製遊戲公會 Game Jam、元智大學的遊戲創作社講座到不正經獨立遊戲冒險者增肥聚會請教前輩意見、參加 Meta 台灣 AR 黑客松競賽到國家衛生研究院科普創作競賽……
此外還有參與 TGDF2023、新一代設計展等關注度較高的大型活動
在這些活動的過程當中，我也認識了許多學生或剛畢業的遊戲製作團隊，而無論是在業界前輩或是其它團隊的口中，南台科大的名字總是伴隨著很高的評價，從早些年令人刮目相看的「罪業狂襲」再到在 TSGDF 中演講的「萬悟劫」、「調香師」製作團隊都給了我很深刻的印象，是我想加入貴校的最主要原因。

# 學習使用 AI 工具

我相信基礎與努力才能帶來豐滿的稻穗，但我同樣也相信合理的工具使用能讓成品完成的更快更好，而 AI 就是其中一項
目前我正在學習 Stable Diffusion (AI 算圖) 與類 Text Generation Webui (文本生成) 的 AI 使用
Stable Diffusion 方面，我正在學習使用組合與試驗各種提示詞，希望能組合出一套提示字 / 反題字並結合 Lora 來算出風格統一的圖，希望能在未來擺脫 Google Colab 的制約，在本地端學習與訓練 Lora，並組件一套合理高效的工作流。
文本生成 AI 的方面，由於我的電腦根本塞不下任何的 LLM (大型語言模型)，我便退而求其次，轉而在角色對話類型的網站上練習製作角色卡，其中我主要活動的便是在 aisekai.ai 上，這是一個類似於 charater.ai 但沒有 filter 的網站，因此我能夠為角色卡制定更多有意思的細節，在未來我希望能夠研發出能使語言模型能夠將自身的 description 疊代的技術，並將這個技術在遊戲中做出讓人感到有靈魂、有記憶、會時刻變化的 NPC。

# 未來規劃

在未來的規劃中，我希望可以在南臺的良好氣氛中建立一支獨立遊戲團隊，並利用學校的師資與環境來挑戰使用 Unreal 引擎來製作遊戲。
透過參與放視、新一代、巴哈姆特 ACG 創作大賽等方式來增加遊戲的曝光度，並力求在畢業後的兩年內能夠將團隊的資金流運轉起來並成立獨立遊戲工作室。
若運營不幸失敗了，我將努力爭取進入微晶、網銀等大廠工作的機會，並運用我同時具備美術與程式的能力應徵技術美術的工作，並嘗試轉為製作人。

while(Moveable==true){gameObject.GetComponent<Body>().MakingGames();}
在築夢的這條路上走了十幾年的我，除了具備了相當的遊戲製作能力與經驗外， 我更有著十分堅毅的意志與強大的執行力，在我的 Discord 中，有一個頻道記錄著每天我做了哪些事情，而我可以很自豪的說，這個紀錄自從 2019 年的 7 月 27 號起至今為止，一天也沒有停止記錄過。

所以，南臺科大的教授們，您們給我一個環境和優秀的學習資源，我則回報給您們一整支傑出校友的團隊，[We deal?].rainbow
