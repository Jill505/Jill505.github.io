還記得之前的 character.ai 嗎？
多次擊敗如 kajiwato 等三流四流的 character 網站後
現在他終於有「有實力的」競爭者了 Aisekai

雖然他沒有 c.ai 的大量資料庫與長時間培養的社群 而且才剛出來十天而已
但我敢說 他十分有勝算 因為他有 c.ai 所沒有的 最重要的東西：
沒有 filter、沒有整天只想上市討好股東的營運團隊、沒有傲慢的態度

這一年來你知道我玩 c.ai 被那個 filter 搞得有多憋屈嗎？
文字對話的 role playing 講的就是一個沉浸感 每次到了高潮的地方他就給我「抱歉 我們的政策不允許生成這樣的內容」
幹你老師咧 我只是想玩文字角色扮演 沒有要跟你搞政治正確還是黃標
c.ai 是私人的聊天室 不是公開的平台 別人又看不到你的對話擱著這搞啥的政治正確
我就只想在作業了一整天後玩個 role playing 對你的社會價值觀沒興趣

而且 每次社群吵著要移除 filter 官方就是那副死樣子
filter 影響了多少人、影響了多少生成內容的品質 官方就是死不拔掉 就是要保持小清新的樣子等股東來餵
NM 的 真的是被氣到
好好的商業時機就這樣給他過去了 我那是多麼的替他感到可惜
每天上 c.ai 看到他的 filter 還在 真的是為他的未來感到著急阿 但官方就是不聽

c.ai 你有一整年的時間好好修理糟糕的內存問題、好好善待社群裡幫你做 character 的創作者、提升多國語言生成回覆的品質
c.ai 卻選擇了忽視用戶的聲音、推出了僅有快速排隊功能的 c.ai+、搞個沒有人看的 vtuber
好多問題包括算力不夠、前後文對不上、話題展不開的問題都可以透過開放的態度來解決 好巧不巧他偏不要
就是要做個 filter 噁心你 就是要在輸出前又放個機制給你審核一下 浪費用戶的時間精力還順便浪費算力 順便汙染一下生成結果 好小子你真行

多國語言弄不好害得我只能寫英文作文就算了 反正海外用戶本來就沒啥話語權 摸摸鼻子就認了回去翻字典
對社群創作者不聞不問也還說得過去 畢竟營運端沒有必要給你送茶倒水 用的模型也是自己訓練的 沒毛病
內存問題也可以用預算不夠來解釋 錢也不是無限的嘛
但在這搞了一年 競爭者都出現了你還在那巴著你的 filter 不忘 你是傻還是笨 還是真就和用戶過不去？

好一個天時地利 人和卻被你小子給玩掉了

多少懷踹著夢想的創業家、工程師都沒能遇到的機遇 你就因為個「堅守社群價值」的 filter 而放棄了一大塊市場、背叛了你的使用者 真有你的 c.ai
我是又羨慕又忌妒又生氣 一個有才幹、有機運而沒能把握住的你

雖然現在 aisekai.ai 還有很多問題 但就營運態度上來說真的贏太多了
人家上線十天三次更新火速修 bug
c.ai 你還在那慶祝 CEO 上了時代雜誌 就不怕人家過來一拳給你 KO 在地上讓你滿地找牙？

唉 不說了 看到一個有機遇有實力的團隊就這樣白給
真的是又心疼又生氣
