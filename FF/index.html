# 【KGJ 2025 - 誰的心靈需要課程】羊電瘋 sheep go crazy


羊電瘋 itch

羊電瘋 itch載點

羊電瘋 介紹(點開)

========================
畫面上方有一個房門，通往小黑屋。
將學生收服後，他們會跟跟你的步伐。此時你必須將學生帶到小黑屋門口附近，等待數秒，把學生關進小黑屋後，才算是洗腦成功。
名次的計算是根據洗腦三十名學生的排序

========================
在這個遊戲，你是一個心靈導師。
你必須洗腦學生，讓他們被你的心靈力給感染。你可以對學生點左鍵來發射心靈能量，連點多次就能讓學生洗腦成功。
你的目標是：取得最大業績，洗腦最多的學生，因此打敗其他三個老師！！！

# 【GGJ 2025 - bubble 泡泡】泡泡糖派對

泡泡糖派對載點(GGJ官網)

泡泡糖派對 GGJ載點

遊戲介紹(點開)

========================
GumBoomer 是一款至多可四人遊玩的派對遊戲，玩家將從場地四角朝中央發射泡泡，並讓泡泡在中心合併成更大的泡泡。
遊戲的目的是在保證自己不讓中間泡泡達到極限大小炸掉的同時，發射越多、越大的泡泡，來取得更高的分數！

操作方式：

  • 每個玩家操作一個鍵盤按鍵 (Q P C M)
  • 長按按鍵蓄力泡泡
  • 放開按鍵射出泡泡
  • 短按按鍵使用道具

※ 警報響起時獲得更高分數加成

道具介紹：

  • 史萊姆（緩速） -> 減速所有敵方玩家射出的泡泡
  • 冰塊（冰凍） -> 讓所有敵方玩家暫時無法行動
  • 閃電（加速） -> 己方玩家的泡泡蓄力速度提升

========================
