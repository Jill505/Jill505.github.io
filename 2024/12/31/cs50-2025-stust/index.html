# 開頭講廢話

欸嗨各位同學們

你們喜歡代碼嗎？
沒錯，我聽到你們的尖叫聲了
想必當然是歡喜的尖叫，而不是痛苦的尖叫對吧... 對吧？

那你們一定會喜歡 CS50！

CS50 是哈佛大學開設的計算機科學入門課程，也是世界上最知名、最受歡迎的線上課程之一！
重點是！完全免費，而且對初學者超級友好！

你可能會想：「我完全沒寫過程式，這真的適合我嗎？」
放心，CS50 的設計初衷就是讓零基礎的你也能輕鬆入門！
老師講解風趣幽默，還有大量的實作練習和專案讓你邊學邊做，從入門到進階，一步步帶你走進程式的世界！

# 為什麼 CS50 對新手友好？

  • 從最簡單的 C 語言開始，循序漸進進入 Python、JavaScript 等實用語言
  • 每堂課都有大量範例與實作，直接寫程式、直接驗證，學習效果超高
  • 重視實作與創造，讓你從理論到實戰，一步步打造屬於自己的專案

# 在 CS50，你將會學到：

  • 程式設計基礎：從 C 語言到 Python，再到 JavaScript，扎實你的基本功
  • 演算法和資料結構：訓練你的邏輯思維，提升解決問題的能力
  • 網頁開發：HTML、CSS、JavaScript 前後端都給你來一點
  • 遊戲開發：學到的技能完全可以用來開發自己的遊戲，從簡單的小品到大型專案都不成問題
  • 人工智慧（AI）和資料分析：幫助你跟上最前沿的科技趨勢

# 對於想要做遊戲的同學們：

CS50 不只是教你寫程式，還會讓你用程式去創造東西！
你可以做出屬於自己的小遊戲，或是參與更大型的開發專案！
老師還會帶你做出經典遊戲，如「貪吃蛇」等，學習遊戲邏輯的同時，還能感受到開發的成就感！

不管你是想成為專業程式設計師 代碼的狗，還是只是想做出一款屬於自己的遊戲，CS50 都是你的最佳選擇！
準備好了嗎？
一起加入 CS50 的行列，開啟你的程式設計之旅吧！

  • 課程完成後會發放證書，完全免費！
